Minimum Requirements:
=====================
- 16 GB of RAM *8 will be allocated for Docker)

Install required Software:
==========================
* Git with Git-bash (https://git-scm.com/downloads)
* Docker for Windows
* GDAL for widnows: https://trac.osgeo.org/gdal/wiki/OSGeo4W
* c++
* Gzip : http://gnuwin32.sourceforge.net/packages/gzip.htm , make sure to add the location of gzip.exe to the windoes PATH variable
* Python 3.7 (https://www.python.org/ftp/python/3.7.2/python-3.7.2.exe)
* Python 3 dependencies:
python3 -m pip install logger docker progressbar2 requests argparse httplib2 apiclient boto3 gdal

no need to pip install the following as they are already included in the default install of python3.7 : json zipfile tarfile re csv math 

Make sure you have the following python3(!) modules installed. 
To list all modules installed, you can use the python command: >>> help("modules")
